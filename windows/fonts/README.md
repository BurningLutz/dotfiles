  - [Italic](https://github.com/ryanoasis/nerd-fonts/raw/master/patched-fonts/CodeNewRoman/Italic/complete/Code%20New%20Roman%20Italic%20Nerd%20Font%20Complete%20Mono%20Windows%20Compatible.otf)
